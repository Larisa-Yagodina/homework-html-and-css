Что такое CSS

CSS — это каскадная таблица стилей. Каскад в данном случае означает одновременное применение разных стилевых правил к элементам документа (чтобы не было конфликтов, введены приоритеты). Это язык, который отвечает за описание внешнего вида HTML-документа. CSS-файл — это файл с расширением .css, где прописываются правила оформления всех документов сайта.

Селектор - способ задать, к каким элементом будет применяться тот или иной стиль. Какие бывают селекторы:

  • тэги;
  • классы - class;
  • идентификаторы - id;

Какие бывают свойства:

  • фон - background;
  • цвет шрифта - color;
  • отступы - margin, button;
  • граница - border;
  • и другие.
